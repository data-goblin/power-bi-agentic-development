# v26.26.1

Hotfix. Refreshes the `useful-stuff/status-lines/` statusline to the current build; no plugin behavior changes.

**Statusline**

- New `07-cost.sh` segment: a marginal `$` spend meter that stays hidden on a subscription and only surfaces when spend actually matters (a rate window over 100%, or fast mode on). It meters from the start of the billable stint, not the inflated cumulative session estimate.
- Git segment gains a behind/unpushed count next to the branch, driven by a non-blocking background `git fetch` (TTL-cached). The LOC delta and untracked-line count are background-refreshed too, so large or heavily-changed trees no longer stall the render and blank the line.
- Portable `_timeout` fallback for stock macOS (no coreutils) and a shared `_mtime` helper; worktree sessions tint the cwd and show the worktree name after the branch.
- README documents the new segment and the behind/unpushed count.
